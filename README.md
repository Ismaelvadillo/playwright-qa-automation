# Playwright QA Automation

## Overview

This project contains automated UI tests developed with Playwright using JavaScript.

The objective is to demonstrate knowledge of web test automation, end-to-end testing, assertions, and test organization following QA best practices.

## Technologies

- Playwright
- JavaScript
- Node.js

## Features

- Automated login testing
- Navigation between pages
- UI element validation
- Assertions
- End-to-End test scenarios

## Project Structure

```
tests/
pages/
playwright.config.js
```

## Installation

```bash
npm install
```

## Run all tests

```bash
npx playwright test
```

## Generate HTML Report

```bash
npx playwright show-report
```

## Skills Demonstrated

- UI Automation
- End-to-End Testing
- Test Assertions
- Test Organization
- Playwright Framework
- Git Version Control
