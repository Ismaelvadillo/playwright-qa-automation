# Playwright QA Automation

## This repository was created as part of my QA training to practice automated web testing using Playwright and JavaScript.

This project contains automated UI tests developed with Playwright using JavaScript.

This project demonstrates practical experience with web test automation using Playwright, including end-to-end testing, assertions and test organization following QA best practices.

## Technologies

- Playwright
- JavaScript
- Node.js

## Features

- Automated login testing
- Navigation between pages
- UI element validation
- Assertions
- End-to-End test scenarios

## Project Structure

```
tests/
pages/
playwright.config.js
```

## Installation

```bash
npm install
```

## Run all tests

```bash
npx playwright test
```

## Generate HTML Report

```bash
npx playwright show-report
```
